자주 묻는 질문 – 텍스트 비교 도구

이 텍스트 비교 도구는 어떻게 작동하나요?
두 텍스트 블록을 즉시 비교하여 브라우저 내에서 색상으로 차이점을 강조 표시합니다.
내 텍스트는 어디에 저장되나요?
전혀 저장되지 않습니다. 모든 처리는 로컬 브라우저에서 이루어집니다.
강조 색상은 무엇인가요?
녹색은 추가된 텍스트, 빨간색은 삭제된 텍스트를 나타내며, 변경되지 않은 텍스트는 평소대로 표시됩니다.
이 도구를 어디에 사용할 수 있나요?
코드 리뷰, 논문 비교, 버전 추적, 교정 편집 등 모든 텍스트 비교 작업에 사용할 수 있습니다.
이 도구는 무료인가요?
예. 이 도구는 100% 무료이며 브라우저 기반으로 로그인이나 회원 가입이 필요 없습니다.
