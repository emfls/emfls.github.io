자주 묻는 질문 (FAQ)

1. 투자이민 비자 신청 시 한국에서 먼저 준비해야 할 서류는 무엇인가요?

답변: 여권, 재정증명(투자금 출처), 사업 계획서(필요시), 건강검진, 무범죄 증명서 등이 기본적으로 필요합니다. 자세한 국가별 목록은 해당 국가 카드 내 '필수 제출 서류' 항목과 공식 사이트 링크를 참조해주세요.

2. 투자금 출처 증빙은 어떤 방식으로 제출해야 하나요?

답변: 은행 잔고 증명, 세무 신고 내역, 주식/부동산 매매 계약서 등으로 자금의 합법적 출처를 증명해야 합니다. 국가별 세부 서류는 해당 국가 카드 내 '필수 제출 서류'와 공식 사이트 링크를 통해 확인하세요.

3. 투자이민 승인 후 실제 이주까지 걸리는 평균 소요 기간은?

답변: 각 국가 카드에 명시된 '소요 기간'을 참고하시기 바랍니다. 예: 미국 EB-5 약 18~24개월, 캐나다 Start-up Visa 약 12~16개월. 더 정확한 정보는 공식 사이트 링크를 통해 확인해주세요.

4. 가족(배우자·자녀) 동반 신청은 어떻게 하나요?

답변: 대부분의 국가에서 주 신청자 승인 시 배우자 및 미성년 자녀의 동반 신청이 가능합니다. 가족관계 증명, 재정능력 증명 등이 필요하며, 국가별 상세 내용은 해당 카드 내 '가족 동반 옵션'과 공식 사이트에서 확인하세요.

5. 투자이민 시 연간 체류 의무는 어떻게 되나요?

답변: 각 국가 카드 내 '최소 체류 의무'를 확인해야 합니다. 예: 호주 비자 유효 기간 중 최소 2년 체류, 싱가포르 매년 최소 30일 체류. 의무 미준수 시 비자 갱신 또는 영주권 전환에 불이익이 있을 수 있으니 유의하세요.
