ANDYMAN란 무엇인가?

ANDYMAN은 원래 개발자가 포기한 후 열정적인 커뮤니티가 인수한 탈중앙화, 커뮤니티 중심의 밈 코인입니다[^6_2][^6_3]. 프로젝트는 포기된 컨트랙트, 0% 세금, 소각된 유동성 풀로 운영되어 완전한 투명성과 보안을 보장합니다.

핵심 특징

  • 커뮤니티 인수: 원래 개발자 포기 후 커뮤니티가 프로젝트 인수
  • 포기된 컨트랙트: 완전한 탈중앙화 보장
  • 0% 세금: 거래 시 추가 수수료 없음
  • 소각된 유동성: 유동성 풀 소각으로 보안 강화
  • 밈 문화: 밈 문화를 수용하고 유기적 성장 추구
주요 포인트: ANDYMAN은 충성스럽고 컬트적인 커뮤니티에 의해 구동되며, 장기적 성장과 책임감 있는 투자에 중점을 둡니다. 활발한 커뮤니티 참여와 집단적 의사결정을 통해 유기적 성장을 추구합니다[^6_2][^6_8].

블록체인 기술

ANDYMAN은 이더리움 블록체인 기반의 ERC-20 토큰으로, 완전한 탈중앙화와 투명성을 보장합니다[^6_2].

핵심 기술 요소

  1. 포기된 컨트랙트: 개발자가 컨트랙트 소유권 포기
  2. 소각된 유동성: 유동성 풀 토큰 영구 소각
  3. 0% 세금 구조: 거래 시 추가 수수료 없음
  4. 커뮤니티 거버넌스: 집단적 의사결정 시스템

토큰 공급량

  • 총 공급량: 10억 ANDYMAN[^6_1]
  • 유통 공급량: 약 9.63억 ANDYMAN[^6_1][^6_2]
  • 최대 공급량: 10억 ANDYMAN[^6_1]
  • 토큰 표준: ERC-20 (이더리움)

토큰 분배

  • 커뮤니티 (100%): 모든 토큰이 커뮤니티 소유
  • 개발팀 (0%): 개발팀 할당 없음
  • 유동성 풀: 소각되어 영구 잠금
  • 마케팅: 커뮤니티 주도 마케팅

유틸리티
