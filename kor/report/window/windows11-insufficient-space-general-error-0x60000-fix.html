Windows 11 0x60000 INSUFFICIENT_SPACE_GENERAL 해결 가이드

⚠️ 주의사항: 이 오류는 Windows 시스템에서 일반적인 저장소 공간이 부족할 때 발생하는 오류입니다. 다양한 시스템 작업에서 나타날 수 있으며 포괄적인 공간 관리가 필요합니다.

🔍 오류 개요

0x60000 INSUFFICIENT_SPACE_GENERAL는 Windows 11에서 시스템 전반적으로 저장소 공간이 부족할 때 발생하는 일반적인 오류입니다. 이 오류는 주로 다음과 같은 원인으로 발생합니다:

  • 시스템 전체 드라이브의 공간 부족
  • 임시 파일 및 캐시 파일 과다 누적
  • 시스템 로그 파일 과다 생성
  • 백업 파일 및 복원 지점 과다 저장
  • 사용자 데이터 무분별한 저장
  • 프로그램 설치 파일 누적
  • 브라우저 캐시 및 다운로드 파일 누적
  • 시스템 업데이트 파일 누적

🛠️ 해결 방법

방법 1: 종합적인 디스크 공간 분석

1 저장소 사용량 분석:
  • 설정 → 시스템 → 저장소
  • 각 드라이브의 사용량 확인
  • 카테고리별 사용량 분석:
    • 시스템 및 예약됨
    • 앱 및 기능
    • 문서
    • 임시 파일
    • 기타
2 디스크 사용량 명령어 확인:
  • 관리자 권한으로 명령 프롬프트 실행
  • 다음 명령어로 상세 분석:
dir C:\ /s | find "bytes"
forfiles /p C:\ /s /m *.* /c "cmd /c echo @path @fsize" | sort /r /+2
wmic logicaldisk get size,freespace,caption

방법 2: 통합 저장소 정리

1 저장소 센스 고급 설정:
  • 설정 → 시스템 → 저장소 → 저장소 센스
  • "저장소 센스 구성" 클릭
  • 모든 정리 옵션 활성화:
    • 임시 파일 자동 삭제: 1일
    • 휴지통 자동 비우기: 14일
    • 다운로드 폴더 정리: 30일
    • OneDrive 로컬 캐시 정리
2 즉시 저장소 정리 실행:
  • "지금 정리" 클릭
  • 정리 진행 상황 모니터링
  • 완료 후 확보된 공간 확인

방법 3: 고급 디스크 정리

1 확장 디스크 정리 실행:
  • 관리자 권한으로 명령 프롬프트 실행
  • 다음 명령어로 고급 정리:
cleanmgr /sageset:65535
cleanmgr /sagerun:65535
  • 모든 정리 옵션 선택
  • 정리 실행 및 완료 대기
2 수동 임시 파일 정리:
  • 다음 명령어들을 순서대로 실행:
del /f /s /q %temp%\*.*
del /f /s /q %windir%\Temp\*.*
del /f /s /q %windir%\Prefetch\*.*
del /f /s /q %windir%\Logs\*.*
del /f /s /q %localappdata%\Temp\*.*

방법 4: 시스템 파일 최적화

1 Windows 구성 요소 정리:
  • 관리자 권한으로 명령 프롬프트 실행
  • 다음 DISM 명령어들을 실행:
DISM /Online /Cleanup-Image /StartComponentCleanup
DISM /Online /Cleanup-Image /StartComponentCleanup /ResetBase
DISM /Online /Cleanup-Image /SPSuperseded
DISM /Online /Cleanup-Image /AnalyzeComponentStore
2 시스템 압축 적용:
  • 시스템 파일 압축으로 공간 절약:
compact /compactos:always
compact /c /s:C:\Windows /i /a
compact /c /s:"C:\Program Files" /i /a

방법 5: 사용자 데이터 관리

1 대용량 파일 검색 및 정리:
  • PowerShell을 관리자 권한으로 실행
  • 대용량 파일 검색:
Get-ChildItem C:\ -Recurse -File | Where-Object {$_.Length -gt 1GB} | Sort-Object Length -Descending | Select-Object FullName, @{Name="Size(GB)";Expression={[math]::Round($_.Length/1GB,2)}} | Format-Table -AutoSize
2 사용자 폴더 정리:
  • 다음 폴더들을 수동으로 정리:
    • 다운로드 폴더 (C:\Users\[사용자명]\Downloads)
    • 문서 폴더의 오래된 파일
    • 바탕화면의 불필요한 파일
    • 사진 및 동영상 폴더의 중복 파일

방법 6: 프로그램 및 앱 정리

1 설치된 프로그램 분석:
  • 설정 → 앱 → 앱 및 기능
  • "크기별 정렬" 선택
  • 대용량 프로그램 확인 및 제거:
    • 사용하지 않는 게임
    • 오래된 개발 도구
    • 중복 설치된 프로그램
    • 평가판 소프트웨어
2 Windows 기능 최적화:
  • 제어판 → 프로그램 → Windows 기능 켜기/끄기
  • 사용하지 않는 기능 비활성화:
    • Internet Explorer 11
    • Windows Media Player
    • XPS 뷰어
    • Hyper-V (필요시)

방법 7: 브라우저 데이터 정리

1 브라우저 캐시 정리:
  • Chrome: Ctrl + Shift + Delete
  • Edge: Ctrl + Shift + Delete
  • Firefox: Ctrl + Shift + Delete
  • 모든 시간 범위 선택하여 삭제:
    • 브라우징 기록
    • 쿠키 및 기타 사이트 데이터
    • 캐시된 이미지 및 파일
    • 다운로드 기록
2 브라우저 확장 프로그램 정리:
  • 사용하지 않는 확장 프로그램 제거
  • 브라우저 프로필 정리
  • 동기화 데이터 최적화

방법 8: 외부 저장소 활용

1 클라우드 저장소 활용:
  • OneDrive 파일 온디맨드 활성화
  • Google Drive, Dropbox 등 추가 클라우드 서비스 활용
  • 로컬 파일을 클라우드로 이동
  • 선택적 동기화 설정으로 로컬 공간 절약
2 외장 저장 장치 활용:
  • 외장 하드 드라이브 연결
  • 대용량 파일 이동:
    • 동영상 파일
    • 사진 아카이브
    • 백업 파일
    • 설치 파일

📋 문제 해결 체크리스트

해결 방법 난이도 소요 시간 공간 확보량 재부팅 필요
디스크 공간 분석 쉬움 15-30분 분석용 아니오
통합 저장소 정리 쉬움 30-60분 10-30GB 아니오
고급 디스크 정리 보통 45-90분 15-40GB 가능
시스템 파일 최적화 어려움 60-180분 5-20GB 예
사용자 데이터 관리 보통 60-120분 대량 아니오
프로그램 정리 쉬움 30-90분 20-100GB 가능
브라우저 데이터 정리 쉬움 15-30분 2-10GB 아니오
외부 저장소 활용 보통 120-300분 대량 아니오

🔧 주요 명령어 참조

공간 분석 명령어

  • wmic logicaldisk get size,freespace,caption: 드라이브 공간 확인
  • dir C:\ /s | find "bytes": 폴더별 크기 확인
  • Get-ChildItem C:\ -Recurse -File | Measure-Object -Property Length -Sum: PowerShell 크기 분석
  • fsutil volume diskfree C:: 디스크 여유 공간 확인

정리 및 최적화 명령어

  • cleanmgr /sageset:65535: 고급 디스크 정리 설정
  • DISM /Online /Cleanup-Image /StartComponentCleanup /ResetBase: 구성 요소 정리
  • compact /compactos:always: 시스템 압축 활성화
  • del /f /s /q %temp%\*.*: 임시 파일 삭제

⚠️ 주의사항

공간 정리 시 주의점

  • 중요한 데이터는 반드시 백업한 후 정리하세요
  • 시스템 파일 압축은 성능에 영향을 줄 수 있습니다
  • 대용량 파일 삭제 전 용도를 확인하세요
  • 브라우저 데이터 삭제 시 로그인 정보가 사라질 수 있습니다
  • Windows 기능 비활성화 시 일부 프로그램에 영향을 줄 수 있습니다
  • 외부 저장소 의존도가 높아질 수 있으니 백업 계획을 세우세요

언제 전문가의 도움이 필요한가?

  • 하드 드라이브 용량이 극도로 제한적인 경우
  • 하드웨어 업그레이드가 필요한 경우
  • 시스템 파일이 심각하게 손상된 경우
  • 데이터 복구가 필요한 경우
  • 기업 환경에서 정책적 제약이 있는 경우

✅ 예방 방법

  • 정기적인 저장소 모니터링 (주간)
  • 자동 정리 기능 활성화 및 최적화
  • 클라우드 저장소 적극 활용
  • 불필요한 파일 즉시 삭제 습관
  • 프로그램 설치 시 신중한 선택
  • 정기적인 백업 및 아카이브
  • 시스템 드라이브 최소 25% 여유 공간 유지

🔍 관련 오류 코드

유사한 저장소 공간 관련 오류들

  • 0x80070070: INSUFFICIENT_DISK_SPACE
  • 0x8007000E: 메모리가 부족합니다
  • 0x80070003: 지정된 경로를 찾을 수 없습니다
  • 0x80070002: 지정된 파일을 찾을 수 없습니다
  • 0x80240002: WU_E_MAX_CAPACITY_REACHED

🛠️ 고급 진단 도구

저장소 분석 도구

  • Windows 저장소 설정: 설정 → 시스템 → 저장소
  • 디스크 정리: cleanmgr
  • 리소스 모니터: resmon - 디스크 활동 모니터링
  • 성능 모니터: perfmon - 저장소 성능 추적

타사 분석 도구

  • TreeSize: 폴더별 상세 용량 분석
  • WinDirStat: 시각적 디스크 사용량 분석
  • CCleaner: 종합 시스템 정리
  • Duplicate Cleaner: 중복 파일 제거

🔄 자동화 스크립트

종합 저장소 정리 스크립트

@echo off
echo 종합 저장소 정리 시작...
echo.
echo 현재 디스크 사용량 확인 중...
wmic logicaldisk get size,freespace,caption
echo.
echo 임시 파일 정리 중...
del /f /s /q %temp%\*.* 2>nul
del /f /s /q %windir%\Temp\*.* 2>nul
del /f /s /q %windir%\Prefetch\*.* 2>nul
del /f /s /q %localappdata%\Temp\*.* 2>nul
echo.
echo 시스템 구성 요소 정리 중...
DISM /Online /Cleanup-Image /StartComponentCleanup /Quiet
echo.
echo 디스크 정리 실행 중...
cleanmgr /sagerun:65535
echo.
echo 정리 완료! 디스크 사용량을 다시 확인하세요.
wmic logicaldisk get size,freespace,caption
pause
  • 위 내용을 메모장에 복사하여 .bat 파일로 저장
  • 관리자 권한으로 실행
  • 정기적으로 실행하여 공간 관리

이 가이드를 통해 Windows 일반적인 저장소 공간 부족 문제가 해결되기를 바랍니다. 지속적인 저장소 관리로 시스템 성능을 최적화하시기 바랍니다.
