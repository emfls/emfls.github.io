🚀 방법 1: Media Creation Tool 사용 (초보자 추천)

1단계

Media Creation Tool 다운로드

✅ 가장 안전하고 쉬운 방법: Microsoft에서 공식 제공하는 도구로 초보자도 실수 없이 설치 미디어를 만들 수 있습니다.

다운로드 과정:

  1. 웹 브라우저(크롬, 엣지 등)를 실행합니다
  2. 주소창에 microsoft.com/software-download/windows11 입력
  3. "Windows 11 설치 미디어 만들기" 섹션 찾기
  4. "지금 도구 다운로드" 버튼 클릭
  5. 다운로드 폴더에 MediaCreationToolW11.exe 파일 확인
📸 화면 예시:
Microsoft 공식 사이트 → "Windows 11 다운로드" → "지금 도구 다운로드" 버튼

✅ 1단계 완료 확인




2단계

USB 드라이브 준비 및 연결

⚠️ 데이터 손실 주의: USB 드라이브의 모든 데이터가 삭제됩니다. 중요한 파일이 있다면 미리 백업하세요.

USB 드라이브 선택 기준:

  • 용량: 최소 8GB, 권장 16GB 이상
  • 속도: USB 3.0 이상 (파란색 포트)
  • 브랜드: 삼성, 샌디스크, 킹스톤 등 신뢰할 수 있는 브랜드
  • 상태: 정상 작동하는 USB (오류 없음)

USB 연결 및 확인:

  1. USB 드라이브를 컴퓨터에 연결
  2. 파일 탐색기(Windows + E) 열기
  3. "이 PC"에서 USB 드라이브 확인
  4. USB 드라이브 우클릭 → "속성"으로 용량 확인
  5. 필요시 USB 드라이브 포맷 (FAT32 또는 NTFS)
# PowerShell로 USB 드라이브 확인 (고급 사용자용)
Get-WmiObject -Class Win32_LogicalDisk | Where-Object {$_.DriveType -eq 2}
# 결과에서 USB 드라이브 정보 확인

✅ 2단계 완료 확인




3단계

Media Creation Tool 실행

💡 관리자 권한 필요: 시스템 파일에 접근하기 위해 관리자 권한으로 실행해야 합니다.

도구 실행 과정:

  1. 다운로드한 MediaCreationToolW11.exe 파일 찾기
  2. 파일 우클릭 → "관리자 권한으로 실행" 선택
  3. 사용자 계정 컨트롤(UAC) 창에서 "예" 클릭
  4. 도구 로딩 대기 (약 10-30초)
  5. 라이선스 조건 화면에서 "동의" 클릭
📸 화면 예시:
파일 우클릭 메뉴 → "관리자 권한으로 실행" → UAC 창 "예" → 라이선스 "동의"

초기 설정 선택:

  • 언어: 한국어 (기본값)
  • 에디션: Windows 11 (권장)
  • 아키텍처: 64비트 (x64)

✅ 3단계 완료 확인




4단계

설치 미디어 생성 옵션 선택

옵션 선택 화면:

  1. "다른 PC용 설치 미디어 만들기" 선택
  2. "다음" 버튼 클릭
  3. 언어, 에디션, 아키텍처 설정 확인
  4. 필요시 "이 PC에 권장되는 옵션 사용" 체크 해제 후 수정

⚠️ 옵션 선택 주의사항

  • 언어: 한국어 선택 (다른 언어 선택 시 한글 지원 안됨)
  • 에디션: Windows 11 (Home/Pro 구분 없음)
  • 아키텍처: 64비트 선택 (32비트는 지원 안됨)

미디어 유형 선택:

  • USB 플래시 드라이브 선택 (권장)
  • ISO 파일 옵션은 고급 사용자용

✅ 4단계 완료 확인




5단계

USB 드라이브 선택 및 생성 시작

USB 드라이브 선택:

  1. 연결된 USB 드라이브 목록에서 올바른 드라이브 선택
  2. 드라이브 이름과 용량을 확인하여 정확한 USB 선택
  3. "다음" 버튼 클릭
  4. 데이터 삭제 경고 메시지에서 "확인" 클릭
🚨 마지막 확인: 선택한 USB 드라이브가 맞는지 다시 한 번 확인하세요. 잘못된 드라이브를 선택하면 중요한 데이터가 삭제될 수 있습니다.

다운로드 및 생성 과정:

  • 1단계: Windows 11 다운로드 (약 4-5GB)
  • 2단계: 다운로드 파일 검증
  • 3단계: USB 드라이브 포맷
  • 4단계: 설치 파일 USB로 복사
  • 5단계: 부팅 가능한 USB 생성

💡 예상 소요 시간

  • 고속 인터넷 + USB 3.0: 15-30분
  • 일반 인터넷 + USB 2.0: 45-60분
  • 느린 인터넷: 1-2시간

✅ 5단계 진행 확인




6단계

생성 완료 및 검증

생성 완료 확인:

  1. "USB 플래시 드라이브 준비 완료" 메시지 확인
  2. "마침" 버튼 클릭
  3. 파일 탐색기에서 USB 드라이브 내용 확인
  4. 주요 파일들이 정상적으로 생성되었는지 확인

USB 내용 확인사항:

파일/폴더명 유형 용도 필수 여부
setup.exe 실행 파일 Windows 설치 프로그램 ✅ 필수
sources 폴더 폴더 설치 파일들 ✅ 필수
boot 폴더 폴더 부팅 파일들 ✅ 필수
efi 폴더 폴더 UEFI 부팅 파일 ✅ 필수
autorun.inf 설정 파일 자동 실행 설정 ⚪ 선택

✅ 생성 성공 확인 방법

  • USB 용량이 4-5GB 정도 사용됨
  • 위 표의 필수 파일들이 모두 존재
  • USB 드라이브 이름이 "ESD-USB" 또는 "CCSA_X64FRE_KO-KR_DV9"로 변경됨
  • 다른 컴퓨터에서 USB 부팅 테스트 가능

🛠️ 방법 2: Rufus 사용 (중급자용)

대안

Rufus로 더 빠른 USB 생성

💡 Rufus 장점: Media Creation Tool보다 빠르고 더 많은 옵션을 제공합니다. 하지만 설정이 복잡하여 중급자 이상에게 권장합니다.

Rufus 사용 과정:

  1. Rufus 다운로드: rufus.ie에서 최신 버전 다운로드
  2. ISO 파일 준비: Microsoft에서 Windows 11 ISO 다운로드
  3. Rufus 실행: 관리자 권한으로 실행
  4. 설정 구성: 아래 권장 설정 적용
  5. 생성 시작: "시작" 버튼 클릭

Rufus 권장 설정:

설정 항목 권장 값 설명
장치 USB 드라이브 선택 대상 USB 드라이브
부팅 선택 디스크 또는 ISO 이미지 Windows 11 ISO 파일
이미지 옵션 Standard Windows 11 installation 표준 설치
파티션 구성 GPT UEFI 시스템용
대상 시스템 UEFI (non CSM) 최신 시스템
파일 시스템 FAT32 UEFI 호환

⚠️ Rufus 사용 시 주의사항

  • 잘못된 설정으로 부팅 불가능한 USB 생성 가능
  • GPT/MBR, UEFI/Legacy 설정을 정확히 선택해야 함
  • 일부 컴퓨터에서 호환성 문제 발생 가능

문제 원인 해결 방법 예방법

✅ USB 부팅 테스트 방법

생성된 USB 정상 작동 확인

  1. BIOS/UEFI 진입: 컴퓨터 부팅 시 F2, F12, Del 키 등
  2. 부팅 순서 변경: USB를 첫 번째 부팅 장치로 설정
  3. 설정 저장: F10 키로 저장 후 재부팅
  4. Windows 설치 화면: 정상적으로 나타나는지 확인
  5. 언어 선택: 한국어가 정상적으로 표시되는지 확인
