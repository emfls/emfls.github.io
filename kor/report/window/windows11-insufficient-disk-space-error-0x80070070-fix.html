Windows 11 0x80070070 INSUFFICIENT_DISK_SPACE 해결 가이드

⚠️ 주의사항: 이 오류는 Windows 시스템에서 디스크 공간이 부족할 때 발생하는 오류입니다. 업데이트, 설치, 파일 작업 등 다양한 상황에서 나타날 수 있습니다.

🔍 오류 개요

0x80070070 INSUFFICIENT_DISK_SPACE는 Windows 11에서 작업을 수행하기 위한 충분한 디스크 공간이 없을 때 발생하는 오류입니다. 이 오류는 주로 다음과 같은 원인으로 발생합니다:

  • 시스템 드라이브(C:)의 여유 공간 부족
  • Windows Update 설치를 위한 공간 부족
  • 임시 파일 누적으로 인한 공간 부족
  • 시스템 복원 지점 과다 생성
  • 휴면 파일(hiberfil.sys) 크기 과다
  • 페이지 파일(pagefile.sys) 크기 과다
  • Windows.old 폴더 누적
  • 사용자 파일(문서, 사진, 동영상) 과다 저장

🛠️ 해결 방법

방법 1: 디스크 정리 도구 사용

1 기본 디스크 정리 실행:
  • Windows + R 키를 누르고 cleanmgr 입력
  • 시스템 드라이브(C:) 선택
  • 다음 항목들을 체크하여 삭제:
    • 임시 파일
    • 시스템 오류 메모리 덤프 파일
    • Windows Update 정리
    • 다운로드 폴더
    • 휴지통
    • 임시 인터넷 파일
2 시스템 파일 정리:
  • 디스크 정리 창에서 "시스템 파일 정리" 클릭
  • 추가로 다음 항목들 체크:
    • 이전 Windows 설치 (Windows.old)
    • Windows 업그레이드 로그 파일
    • 시스템 압축 파일
    • Windows Defender 바이러스 백신
  • "확인" 클릭하여 정리 실행

방법 2: 저장소 센스 활성화

1 저장소 센스 설정:
  • 설정 → 시스템 → 저장소
  • "저장소 센스" 활성화
  • "저장소 센스 구성" 클릭
  • 자동 정리 주기를 "매주" 또는 "매월"로 설정
2 임시 파일 자동 정리 설정:
  • "임시 파일" 섹션에서 다음 설정:
    • "앱에서 사용하지 않는 임시 파일 삭제": 1일
    • "다운로드 폴더에서 파일 삭제": 30일
    • "휴지통에서 파일 삭제": 30일
  • "지금 정리" 클릭하여 즉시 정리

방법 3: 대용량 파일 및 불필요한 프로그램 제거

1 대용량 파일 찾기:
  • 설정 → 시스템 → 저장소
  • 각 카테고리 클릭하여 용량 확인:
    • 앱 및 기능
    • 문서
    • 사진
    • 음악
    • 동영상
    • 기타
2 불필요한 프로그램 제거:
  • 설정 → 앱 → 앱 및 기능
  • "크기별 정렬" 선택하여 대용량 앱 확인
  • 사용하지 않는 프로그램 선택 후 "제거"
  • 다음과 같은 프로그램들 확인:
    • 게임
    • 오래된 소프트웨어
    • 중복 설치된 프로그램
    • 평가판 소프트웨어

방법 4: 시스템 복원 지점 관리

1 시스템 복원 설정 확인:
  • 제어판 → 시스템 및 보안 → 시스템 → 시스템 보호
  • 시스템 드라이브 선택 후 "구성" 클릭
  • 현재 디스크 사용량 확인
  • 최대 사용량을 5-10%로 조정
2 이전 복원 지점 삭제:
  • "삭제" 버튼 클릭
  • "가장 오래된 복원 지점을 제외한 모든 복원 지점 삭제" 선택
  • 또는 명령 프롬프트에서 다음 명령어 실행:
vssadmin delete shadows /all /quiet
wmic shadowcopy delete

방법 5: 휴면 파일 및 페이지 파일 최적화

1 휴면 파일 크기 조정:
  • 관리자 권한으로 명령 프롬프트 실행
  • 현재 휴면 파일 크기 확인:
powercfg /hibernate /size 50
  • 또는 휴면 기능 완전 비활성화 (권장하지 않음):
powercfg /hibernate off
2 페이지 파일 크기 조정:
  • 제어판 → 시스템 → 고급 시스템 설정
  • "성능" 섹션의 "설정" 클릭
  • "고급" 탭 → "가상 메모리" 변경
  • "모든 드라이브에 대한 페이징 파일 크기 자동 관리" 체크 해제
  • 시스템 드라이브 선택 후 "사용자 지정 크기":
    • 초기 크기: RAM 크기 × 1
    • 최대 크기: RAM 크기 × 1.5

방법 6: 사용자 폴더 이동

1 사용자 폴더를 다른 드라이브로 이동:
  • 문서, 사진, 동영상, 음악 폴더 우클릭
  • "속성" → "위치" 탭
  • "이동" 버튼 클릭
  • 다른 드라이브(D:, E: 등)의 폴더 선택
  • "확인" 클릭하여 파일 이동
2 OneDrive 동기화 폴더 변경:
  • OneDrive 아이콘 우클릭 → "설정"
  • "계정" 탭 → "폴더 선택"
  • 동기화할 폴더 선택 최소화
  • 또는 OneDrive 폴더를 다른 드라이브로 이동

방법 7: WinSxS 폴더 정리

1 구성 요소 저장소 정리:
  • 관리자 권한으로 명령 프롬프트 실행
  • 다음 명령어들을 순서대로 실행:
DISM /Online /Cleanup-Image /StartComponentCleanup
DISM /Online /Cleanup-Image /StartComponentCleanup /ResetBase
DISM /Online /Cleanup-Image /SPSuperseded
2 Windows Update 캐시 정리:
  • 다음 명령어로 업데이트 캐시 정리:
net stop wuauserv
rd /s /q %windir%\SoftwareDistribution\Download
net start wuauserv

방법 8: 디스크 압축 및 중복 제거

1 NTFS 압축 사용:
  • 시스템 드라이브(C:) 우클릭 → "속성"
  • "이 드라이브를 압축하여 디스크 공간 절약" 체크
  • "확인" 클릭 (시간이 오래 걸림)
  • 또는 명령어로 특정 폴더 압축:
compact /c /s:C:\Users /i /a
compact /c /s:C:\Program Files /i /a
2 중복 파일 제거:
  • 타사 도구 사용 (Duplicate Cleaner, CCleaner 등)
  • 수동으로 중복 파일 검색:
    • 다운로드 폴더
    • 문서 폴더
    • 사진 폴더
    • 동영상 폴더

📋 문제 해결 체크리스트

해결 방법 난이도 소요 시간 공간 확보량 재부팅 필요

저장소 센스 활성화 쉬움 10-20분 2-10GB 아니오
불필요한 프로그램 제거 쉬움 30-60분 10-50GB 가능
시스템 복원 지점 관리 보통 10-20분 5-15GB 아니오
휴면/페이지 파일 최적화 보통 15-30분 5-20GB 예
사용자 폴더 이동 어려움 60-180분 대량 아니오
WinSxS 폴더 정리 어려움 30-90분 3-10GB 예
디스크 압축 어려움 120-600분 10-30% 아니오

🔧 주요 명령어 참조

디스크 정리 명령어

  • cleanmgr: 디스크 정리 도구
  • cleanmgr /sageset:1: 디스크 정리 설정
  • cleanmgr /sagerun:1: 자동 디스크 정리 실행
  • del /f /s /q %temp%\*.*: 임시 파일 삭제

시스템 최적화 명령어

  • DISM /Online /Cleanup-Image /StartComponentCleanup: 구성 요소 정리
  • vssadmin delete shadows /all: 섀도우 복사본 삭제
  • powercfg /hibernate off: 휴면 기능 비활성화
  • compact /c /s:C:\ /i: 파일 압축

⚠️ 주의사항

디스크 공간 정리 시 주의점

  • 중요한 파일은 미리 백업하세요
  • 시스템 복원 지점 삭제 시 신중하게 결정하세요
  • 휴면 기능 비활성화 시 빠른 시작 기능이 제한됩니다
  • 페이지 파일 크기 변경 후 시스템 성능을 모니터링하세요
  • 디스크 압축은 시스템 성능에 영향을 줄 수 있습니다
  • 사용자 폴더 이동 시 일부 프로그램에서 문제가 발생할 수 있습니다

언제 전문가의 도움이 필요한가?

  • 하드 드라이브 용량이 극도로 작은 경우 (128GB 이하)
  • 하드웨어 업그레이드가 필요한 경우
  • 디스크 오류가 발생한 경우
  • 시스템 파일이 심각하게 손상된 경우
  • 모든 해결 방법을 시도했지만 공간이 부족한 경우

✅ 예방 방법

  • 정기적인 디스크 정리 실행 (월 1회 이상)
  • 저장소 센스 자동 정리 기능 활성화
  • 시스템 드라이브에 최소 20% 여유 공간 유지
  • 불필요한 프로그램 정기적 제거
  • 대용량 파일은 외부 저장소에 보관
  • 클라우드 저장소 활용 (OneDrive, Google Drive 등)
  • 정기적인 파일 정리 및 정돈

💡 추가 팁

  • 디스크 사용량 분석 도구로 공간 사용 현황 파악
  • 외장 하드 드라이브나 USB 드라이브 활용
  • 클라우드 저장소의 선택적 동기화 기능 사용
  • SSD 업그레이드 고려 (더 큰 용량)
  • 정기적인 파일 백업으로 중요 데이터 보호
  • 브라우저 캐시 및 쿠키 정기적 삭제

🔍 관련 오류 코드

유사한 디스크 공간 관련 오류들

  • 0x8007000E: 메모리가 부족합니다
  • 0x80070002: 지정된 파일을 찾을 수 없습니다
  • 0x80070003: 지정된 경로를 찾을 수 없습니다
  • 0x80240002: WU_E_MAX_CAPACITY_REACHED
  • 0x80070005: 액세스가 거부되었습니다
  • 0x8007007E: 지정된 모듈을 찾을 수 없습니다

Windows 내장 도구

  • 저장소 설정: 설정 → 시스템 → 저장소
  • 디스크 정리: cleanmgr
  • 디스크 관리: diskmgmt.msc
  • 리소스 모니터: resmon

타사 도구

  • TreeSize Free: 폴더별 용량 분석
  • WinDirStat: 시각적 디스크 사용량 분석
  • CCleaner: 종합 시스템 정리
  • SpaceSniffer: 실시간 디스크 사용량 분석

🔄 자동화 스크립트

디스크 공간 정리 자동화 스크립트

@echo off
echo 디스크 공간 정리 시작...
echo 임시 파일 삭제 중...
del /f /s /q %temp%\*.*
del /f /s /q %windir%\Temp\*.*
del /f /s /q %windir%\Prefetch\*.*
echo Windows Update 캐시 정리 중...
net stop wuauserv
rd /s /q %windir%\SoftwareDistribution\Download
net start wuauserv
echo 휴지통 비우기...
rd /s /q C:\$Recycle.Bin
echo 구성 요소 저장소 정리 중...
DISM /Online /Cleanup-Image /StartComponentCleanup
echo 디스크 정리 도구 실행...
cleanmgr /sagerun:1
echo 정리 완료!
pause
  • 위 내용을 메모장에 복사하여 .bat 파일로 저장
  • 관리자 권한으로 실행
  • 정기적으로 실행하여 공간 확보
