Windows 11 설치 전 반드시 확인해야 할 5가지

⚠️ 한눈에 보기: Windows 11은 TPM 2.0과 Secure Boot를 필수 요구사항으로 지정합니다. 설치 진행 전, 아래 5가지 체크리스트를 반드시 완료하세요.
1

시스템 최소‧권장 사양 충족 여부

항목 최소 사양 권장 사양
CPU 1 GHz, 2코어, 64-bit 8세대 Intel / Ryzen 2000 ↑
RAM 4 GB 8 GB ↑
저장소 64 GB SSD 256 GB ↑
펌웨어 UEFI + Secure Boot UEFI 2.3.1
TPM TPM 2.0 TPM 2.0 (필수)
GPU DX12 + WDDM 2.0 DX12 Ultimate
PC Health Check 앱을 실행해 “이 PC는 Windows 11을 실행할 수 있습니다” 메시지를 확인하세요.
# PowerShell 빠른 사양 점검
Get-ComputerInfo | Select-Object WindowsProductName, OsVersion, CsProcessors
Get-Tpm | Format-List TpmPresent,TpmReady,TpmVersion
2

TPM 2.0 & Secure Boot 활성화

TPM 2.0 활성화 방법

  1. Win+R → tpm.msc 입력 후 버전 확인
  2. BIOS/UEFI 진입 → Intel PTT / AMD fTPM 설정을 Enabled

Secure Boot 활성화 방법

  1. BIOS에서 CSM(레거시 지원)을 Disabled
  2. Secure Boot 항목을 Enabled
  3. 부팅 모드를 UEFI로 고정
주의! Secure Boot 활성화 시 구형 OS·드라이버는 부팅되지 않을 수 있습니다.
3

하드웨어‧소프트웨어 호환성·드라이버 업데이트

필수 드라이버 최신 버전 확인

  • 그래픽: NVIDIA/AMD/Intel 최신 Game Ready·Adrenalin·ARC 드라이버
  • 칩셋 & 스토리지: Intel RST, AMD Chipset, Samsung/NVMe SSD
  • 네트워크·오디오: 제조사 공식 최신 드라이버

중요 소프트웨어 호환성

  • 안티바이러스/방화벽 프로그램: Windows 11 지원 버전인지 확인
  • 업무용 앱(Adobe, CAD, 개발툴): 최신 패치 적용 후 테스트
4

데이터 백업·복구 시나리오

백업 체크리스트

  • 문서·사진·동영상 등 개인 파일
  • 브라우저 북마크·암호
  • Outlook PST / 이메일 데이터
  • 프로그램 라이선스·시리얼
  • 전체 시스템 이미지 (권장)

5

설치 미디어 준비 & 설치 방법 결정

설치 미디어 만들기

  1. Microsoft “Windows 11 다운로드” 페이지 접속
  2. 미디어 생성 도구 실행 → 16 GB USB 선택
  3. 완료 후 BIOS 부트 순서에서 USB를 1순위
