Windows 11 0x80070002 FILE_NOT_FOUND 해결 가이드

⚠️ 주의사항: 이 오류는 Windows 시스템에서 필요한 파일을 찾을 수 없을 때 발생하는 오류입니다. 시스템 파일 누락이나 경로 오류가 주요 원인입니다.

🔍 오류 개요

0x80070002 FILE_NOT_FOUND는 Windows 11에서 시스템이나 프로그램이 필요한 파일을 찾을 수 없을 때 발생하는 오류입니다. 이 오류는 주로 다음과 같은 원인으로 발생합니다:

  • Windows Update 파일 누락 또는 손상
  • 시스템 파일 경로 오류
  • 레지스트리 키 손상 또는 누락
  • 업데이트 다운로드 중 파일 손실
  • 바이러스나 악성 코드에 의한 파일 삭제
  • 디스크 오류로 인한 파일 손상
  • 부적절한 파일 삭제
  • 시스템 복원 과정에서의 파일 누락

🛠️ 해결 방법

방법 1: 시스템 파일 검사 및 복구

1 SFC 스캔 실행:
  • 관리자 권한으로 명령 프롬프트 실행
  • 다음 명령어 입력:
sfc /scannow
  • 스캔 완료까지 기다리기 (30-90분 소요)
  • 손상된 파일이 발견되면 자동으로 복구
  • 복구 완료 후 시스템 재시작
2 DISM 명령어 실행:
  • SFC 스캔 완료 후 다음 명령어들을 순서대로 실행:
DISM /Online /Cleanup-Image /CheckHealth
DISM /Online /Cleanup-Image /ScanHealth
DISM /Online /Cleanup-Image /RestoreHealth
  • 각 명령어 완료 후 다음 단계로 진행
  • 복구 완료 후 시스템 재시작

방법 2: Windows Update 구성 요소 재설정

1 Windows Update 서비스 중지:
  • 관리자 권한으로 명령 프롬프트 실행
  • 다음 명령어들을 순서대로 실행:
net stop wuauserv
net stop cryptSvc
net stop bits
net stop msiserver
2 업데이트 폴더 이름 변경:
  • 다음 명령어들을 실행하여 기존 폴더 백업:
ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
ren C:\Windows\System32\catroot2 catroot2.old
3 서비스 재시작:
  • 다음 명령어들을 순서대로 실행:
net start cryptSvc
net start bits
net start wuauserv
net start msiserver

방법 3: 누락된 파일 수동 복원

1 오류 메시지에서 파일 경로 확인:
  • 오류 메시지에 표시된 정확한 파일 경로 기록
  • 해당 경로에 파일이 실제로 존재하는지 확인
  • 파일 탐색기에서 숨겨진 파일 표시 활성화
  • 시스템 파일 보호 속성 확인
2 Windows 설치 미디어에서 파일 복사:
  • Windows 11 설치 USB 또는 ISO 파일 준비
  • 설치 미디어를 마운트하거나 연결
  • 누락된 파일을 설치 미디어에서 찾기
  • 관리자 권한으로 파일을 올바른 위치에 복사

방법 4: 레지스트리 수정 및 복구

1 레지스트리 백업:
  • Windows + R 키를 누르고 regedit 입력
  • 파일 → 내보내기로 전체 레지스트리 백업
  • 안전한 위치에 백업 파일 저장
2 손상된 레지스트리 키 복구:
  • 다음 경로로 이동하여 Windows Update 관련 키 확인:
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\WindowsUpdate
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\wuauserv
  • 손상된 키가 있다면 삭제 후 서비스 재시작
  • 필요시 레지스트리 복구 도구 사용

방법 5: 디스크 오류 검사 및 복구

1 디스크 검사 실행:
  • 관리자 권한으로 명령 프롬프트 실행
  • 다음 명령어 실행:
chkdsk C: /f /r
  • 시스템 재시작 시 디스크 검사 실행 여부 묻는 메시지에서 "Y" 입력
  • 재부팅 후 자동으로 디스크 검사 실행
2 추가 드라이브 검사:
  • 다른 드라이브도 동일하게 검사:
chkdsk D: /f /r
chkdsk E: /f /r
  • 모든 드라이브 검사 완료 후 시스템 재시작

방법 6: Windows Update 문제 해결사 실행

1 자동 문제 해결사 실행:
  • 설정 → 시스템 → 문제 해결 → 기타 문제 해결사
  • "Windows Update" 항목에서 "실행" 클릭
  • 문제 해결사가 파일 누락 관련 문제 자동 진단
  • 발견된 문제에 대한 수정 사항 적용
2 고급 문제 해결사 실행:
  • Windows + R 키를 누르고 다음 명령어 입력:
msdt.exe -id WindowsUpdateDiagnostic
  • "고급" 옵션 선택
  • "관리자 권한으로 실행" 체크
  • 모든 권장 수정 사항 적용

방법 7: 바이러스 검사 및 악성 코드 제거

1 Windows Defender 전체 검사:
  • Windows 보안 → 바이러스 및 위협 방지
  • "검사 옵션" → "Microsoft Defender 오프라인 검사" 선택
  • "지금 검사" 클릭하여 재부팅 후 검사 실행
  • 발견된 위협 제거
2 악성 소프트웨어 제거 도구 실행:
  • Microsoft 공식 사이트에서 MSRT(악성 소프트웨어 제거 도구) 다운로드
  • 관리자 권한으로 실행
  • 전체 검사 수행
  • 발견된 악성 코드 제거

방법 8: 시스템 복원 또는 재설치

1 시스템 복원 실행:
  • 제어판 → 시스템 및 보안 → 시스템 → 시스템 보호
  • "시스템 복원" 클릭
  • 오류 발생 이전 날짜의 복원 지점 선택
  • 복원 과정 완료 후 시스템 재시작
2 Windows 재설치 고려:
  • 모든 방법이 실패한 경우 Windows 재설치 고려
  • 중요한 데이터 백업 후 진행
  • Windows 11 설치 미디어 사용
  • "개인 파일 유지" 옵션 선택하여 데이터 보존

🔧 주요 명령어 참조

시스템 복구 명령어

  • sfc /scannow: 시스템 파일 검사 및 복구
  • DISM /Online /Cleanup-Image /RestoreHealth: 시스템 이미지 복구
  • chkdsk C: /f /r: 디스크 오류 검사 및 복구
  • sfc /verifyonly: 시스템 파일 검사만 수행

Windows Update 관련 명령어

  • net stop wuauserv: Windows Update 서비스 중지
  • net start wuauserv: Windows Update 서비스 시작
  • wuauclt /resetauthorization /detectnow: 업데이트 권한 재설정
  • cleanmgr: 디스크 정리 도구

파일 복구 시 주의점

  • 레지스트리 수정 전 복원 지점을 생성하세요
  • 관리자 권한으로 모든 작업을 수행하세요
  • 바이러스 검사 후 시스템 파일 복구를 진행하세요
  • 디스크 검사 중에는 PC를 사용하지 마세요
  • 중요한 데이터는 미리 백업하세요

언제 전문가의 도움이 필요한가?

  • 하드웨어 문제가 의심되는 경우
  • 시스템 파일이 심각하게 손상된 경우
  • 바이러스 감염이 심각한 경우
  • 중요한 업무용 PC에서 문제가 발생한 경우
  • 모든 해결 방법을 시도했지만 문제가 지속되는 경우

✅ 예방 방법

  • 정기적인 시스템 파일 검사 실행 (월 1회)
  • 자동 백업 설정으로 중요 파일 보호
  • 정기적인 바이러스 검사 실행
  • 시스템 복원 지점 정기적 생성
  • 디스크 정리를 통한 공간 확보
  • 신뢰할 수 있는 소프트웨어만 설치
  • Windows Update 자동 설정 유지

💡 추가 팁

  • 파일 누락 오류 발생 시 이벤트 뷰어에서 상세 로그 확인
  • 시스템 파일 보호 기능 활성화 상태 유지
  • 정기적인 하드 디스크 상태 점검
  • 중요한 시스템 파일의 백업 복사본 보관
  • Windows 설치 미디어를 항상 준비해두기
  • 시스템 성능 모니터링으로 문제 조기 발견

🔍 관련 오류 코드

유사한 파일 관련 오류들

  • 0x80070003: 지정된 경로를 찾을 수 없습니다
  • 0x80070005: 액세스가 거부되었습니다
  • 0x8007000E: 메모리가 부족합니다
  • 0x80070020: 다른 프로세스가 파일을 사용 중입니다
  • 0x80070570: 파일 또는 디렉터리가 손상되어 읽을 수 없습니다
  • 0x80004005: 지정되지 않은 오류

🛠️ 고급 진단 도구

파일 시스템 진단 도구

  • 이벤트 뷰어: eventvwr.msc - 파일 관련 오류 로그
  • 시스템 정보: msinfo32 - 시스템 상태 확인
  • 리소스 모니터: resmon - 파일 시스템 활동 모니터링
  • 성능 모니터: perfmon - 시스템 성능 추적

PowerShell 진단 명령어

  • Get-ChildItem -Path C:\ -Recurse -ErrorAction SilentlyContinue: 파일 시스템 검사
  • Test-Path "파일경로": 특정 파일 존재 여부 확인
  • Get-WinEvent -LogName System | Where-Object {$_.Id -eq 7034}: 서비스 오류 로그
  • Get-EventLog -LogName System -EntryType Error: 시스템 오류 로그

📊 파일 누락 오류 진단 체크리스트

단계별 진단 방법

  1. 오류 메시지 분석: 정확한 파일 경로 및 파일명 확인
  2. 파일 존재 여부 확인: 해당 경로에 파일이 실제로 있는지 확인
  3. 파일 권한 확인: 파일 접근 권한 및 소유권 상태
  4. 디스크 상태 확인: 하드 디스크 오류 및 불량 섹터 여부
  5. 시스템 무결성 확인: 시스템 파일 손상 여부
  6. 보안 상태 확인: 바이러스 감염 또는 악성 코드 여부
