Windows 11 설치 전 반드시 확인해야 할 5가지

⚠️ 중요: Windows 11은 이전 버전과 달리 엄격한 시스템 요구사항을 가지고 있습니다. 설치 전 반드시 아래 5가지 사항을 확인하여 원활한 설치와 안정적인 사용을 보장하시기 바랍니다.

🔍 시스템 사양 확인 방법

Windows + R 키를 누르고 다음 명령어들을 실행하여 시스템 정보를 확인하세요:
  • msinfo32 - 전체 시스템 정보
  • dxdiag - DirectX 진단 도구
  • tpm.msc - TPM 관리 콘솔
# PowerShell에서 시스템 정보 확인
Get-ComputerInfo | Select-Object WindowsProductName, TotalPhysicalMemory, CsProcessors
Get-TPM
Get-SecureBootUEFI -Name SetupMode
2

TPM 2.0 및 Secure Boot 활성화

⚠️ 필수 요구사항: TPM 2.0과 Secure Boot는 Windows 11의 필수 보안 기능입니다. 이 기능들이 비활성화되어 있으면 Windows 11을 설치할 수 없습니다.

🔐 TPM 2.0 확인 및 활성화

TPM 상태 확인 방법:
  1. Windows + R 키를 누르고 tpm.msc 입력
  2. TPM 관리 콘솔에서 TPM 버전 확인
  3. "TPM을 사용할 준비가 되었습니다" 메시지 확인

TPM이 비활성화된 경우 활성화 방법:

  1. 컴퓨터 재부팅 후 BIOS/UEFI 설정 진입 (보통 F2, F12, Del 키)
  2. Security 또는 Advanced 메뉴에서 TPM 설정 찾기
  3. TPM 2.0 활성화 (Intel PTT 또는 AMD fTPM)
  4. 설정 저장 후 재부팅

🛡️ Secure Boot 확인 및 활성화

# PowerShell에서 Secure Boot 상태 확인
Confirm-SecureBootUEFI
Get-SecureBootPolicy

Secure Boot 활성화 방법:

  1. BIOS/UEFI 설정에서 Boot 또는 Security 메뉴 이동
  2. Secure Boot 옵션을 "Enabled"로 설정
  3. CSM (Compatibility Support Module) 비활성화
  4. Boot Mode를 UEFI로 설정
주의: Secure Boot 활성화 시 일부 오래된 하드웨어나 운영체제가 부팅되지 않을 수 있습니다. 듀얼 부팅 환경에서는 특히 주의가 필요합니다.
3

하드웨어 호환성 및 드라이버 확인

🖥️ 하드웨어 호환성 검사

PC Health Check 앱 사용:
  • Microsoft 공식 PC Health Check 앱 다운로드
  • "지금 확인" 버튼 클릭하여 호환성 검사
  • 문제점 및 해결 방법 확인

🔧 주요 확인 사항

구성 요소 확인 방법 문제 해결
CPU 호환성 Microsoft 지원 CPU 목록 확인 지원되지 않는 CPU는 업그레이드 필요
그래픽 드라이버 장치 관리자에서 드라이버 버전 확인 최신 WDDM 2.0 이상 드라이버 설치
네트워크 어댑터 Windows 11 호환 드라이버 확인 제조사 웹사이트에서 최신 드라이버 다운로드
오디오 장치 사운드 설정에서 장치 인식 확인 Windows 11 호환 오디오 드라이버 설치
주변기기 프린터, 스캐너 등 호환성 확인 제조사 지원 페이지에서 호환성 확인

📱 필수 소프트웨어 호환성

확인해야 할 소프트웨어:
  • 보안 소프트웨어 - 백신, 방화벽 프로그램
  • 업무용 소프트웨어 - Office, Adobe 제품군
  • 게임 - 주로 사용하는 게임들
  • 전문 소프트웨어 - CAD, 개발 도구 등
# 설치된 프로그램 목록 확인
Get-WmiObject -Class Win32_Product | Select-Object Name, Version
# 또는
Get-ItemProperty HKLM:\Software\Microsoft\Windows\CurrentVersion\Uninstall\* | Select-Object DisplayName, DisplayVersion

데이터 백업 및 복구 계획 수립

⚠️ 필수 작업: Windows 11 설치 과정에서 데이터 손실이 발생할 수 있습니다. 반드시 중요한 데이터를 백업하고 복구 계획을 수립하세요.

💾 백업해야 할 데이터

우선순위별 백업 목록:
  1. 개인 파일 - 문서, 사진, 동영상, 음악
  2. 설정 파일 - 브라우저 북마크, 이메일 설정
  3. 프로그램 설정 - 게임 세이브, 소프트웨어 설정
  4. 라이선스 키 - 소프트웨어 제품키, 인증서
  5. 드라이버 - 특수 하드웨어 드라이버

🔄 백업 방법

백업 방법 장점 단점 권장 용도
외장 하드드라이브 빠른 속도, 대용량 물리적 손상 위험 전체 시스템 백업
클라우드 스토리지 안전성, 접근성 인터넷 속도 의존 중요 문서, 사진
USB 드라이브 휴대성, 저렴함 용량 제한 설정 파일, 드라이버
네트워크 스토리지 자동 백업 가능 초기 설정 복잡 정기적 자동 백업

💿 Windows 11 설치 미디어 생성

🚨 일반적인 설치 문제 및 해결방법

TPM 관련 문제

  • 문제: "이 PC는 Windows 11을 실행할 수 없습니다" 메시지
  • 해결: BIOS에서 TPM 2.0 활성화 또는 CPU의 fTPM/PTT 기능 활성화

Secure Boot 문제

  • 문제: Secure Boot 활성화 불가
  • 해결: CSM 비활성화, UEFI 모드로 부팅, 레거시 옵션 제거

CPU 호환성 문제

  • 문제: 지원되지 않는 CPU
  • 해결: 레지스트리 수정 또는 우회 방법 사용 (권장하지 않음)
