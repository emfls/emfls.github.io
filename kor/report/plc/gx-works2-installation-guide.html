⚠️ 소프트웨어 요구사항

  • 운영체제: Windows 7 SP1 / 8.1 / 10 / 11 (32bit/64bit)
  • Microsoft .NET Framework: 4.0 이상
  • Internet Explorer: 8.0 이상
  • Microsoft Visual C++: 2010 재배포 가능 패키지
  • Windows Installer: 3.1 이상

💡 호환성 정보

  • 지원 PLC: MELSEC-Q, MELSEC-L, MELSEC-FX, MELSEC iQ-R 시리즈
  • 이전 버전 호환: GX Developer 프로젝트 변환 가능
  • 네트워크: Ethernet, USB, RS-232C 통신 지원
  • 언어: 한국어, 영어, 일본어 등 다국어 지원

🔧 설치 전 준비사항

📝 설치 전 체크리스트

⚠️ 중요한 주의사항

  • 기존 버전 제거: 이전 GX Works 버전이 설치되어 있다면 완전히 제거 후 설치
  • 방화벽 설정: Windows 방화벽에서 GX Works2 허용 설정 필요
  • 사용자 계정 제어: UAC 설정이 높을 경우 설치 중 문제 발생 가능
  • 임시 폴더: 충분한 임시 폴더 공간 확보 (최소 1GB)

🛡️ 보안 설정 권장사항

  • 설치 중에만 바이러스 백신 실시간 검사 비활성화
  • Windows Defender 예외 목록에 설치 폴더 추가
  • 방화벽에서 GX Works2 통신 포트 허용
  • 설치 완료 후 보안 프로그램 재활성화

📥 다운로드 방법

미쓰비시 전기 공식 웹사이트 접속

미쓰비시 전기 한국 공식 웹사이트(www.mitsubishielectric.co.kr)에 접속합니다.

제품 정보 → 소프트웨어 메뉴 이동

상단 메뉴에서 "제품 정보" → "FA 제품" → "소프트웨어" 순으로 이동합니다.

GX Works2 검색 및 선택

검색창에 "GX Works2"를 입력하거나 PLC 프로그래밍 소프트웨어 카테고리에서 찾습니다.

회원 가입 및 로그인

다운로드를 위해 회원 가입 후 로그인합니다. 기업 회원 등록이 필요할 수 있습니다.

버전 선택 및 다운로드

최신 버전을 선택하고 라이선스 동의 후 다운로드를 진행합니다.

📦 GX Works2 버전 정보

현재 최신 버전 정보와 다운로드 옵션

  • USB 드라이버: PLC 연결용 드라이버
  • 매뉴얼: 사용 설명서 및 참조 문서
  • 샘플 프로젝트: 학습용 예제 파일
  • 업데이트 파일: 최신 패치 및 업데이트

💡 다운로드 팁

  • 안정적인 인터넷 연결 환경에서 다운로드
  • 다운로드 관리자 사용 권장 (대용량 파일)
  • 파일 무결성 검증을 위한 체크섬 확인
  • 다운로드 완료 후 바이러스 검사 실행

⚙️ 설치 과정

설치 파일 실행

다운로드한 설치 파일을 마우스 우클릭하여 "관리자 권한으로 실행"을 선택합니다.

설치 언어 선택

설치 프로그램이 시작되면 사용할 언어를 선택합니다. (한국어 권장)

라이선스 계약 동의

소프트웨어 라이선스 계약서를 읽고 "동의함"을 선택합니다.

설치 유형 선택

"표준 설치" 또는 "사용자 정의 설치"를 선택합니다. 초보자는 표준 설치 권장.

설치 경로 지정

설치 경로를 지정합니다. 기본 경로(C:\Program Files\MELSOFT\)를 권장합니다.

구성 요소 선택

설치할 구성 요소를 선택합니다. 전체 설치를 권장합니다.

설치 진행

"설치" 버튼을 클릭하여 설치를 시작합니다. 진행률을 확인하며 대기합니다.

설치 완료

설치가 완료되면 "완료" 버튼을 클릭하고 시스템을 재부팅합니다.

🔧 설치 구성 요소 상세

  • GX Works2 Core: 메인 프로그래밍 환경 (필수)
  • Simulator: PLC 시뮬레이션 기능
  • Help Files: 도움말 및 매뉴얼
  • Sample Projects: 예제 프로젝트 파일
  • Communication Drivers: 통신 드라이버
  • Additional Tools: 추가 유틸리티

⚠️ 설치 중 주의사항

  • 설치 중 다른 프로그램 실행 금지
  • 네트워크 연결 안정성 확인
  • 충분한 디스크 공간 확보
  • 설치 로그 파일 보관
  • 설치 실패 시 완전 제거 후 재설치

🔑 라이선스 등록

GX Works2 실행

설치 완료 후 바탕화면의 GX Works2 아이콘을 더블클릭하여 실행합니다.

라이선스 관리자 접속

Help 메뉴 → License Manager를 선택하여 라이선스 관리 창을 엽니다.

라이선스 키 입력

구매 시 제공받은 라이선스 키를 정확히 입력합니다. (대소문자 구분)

온라인 인증

인터넷 연결 상태에서 온라인 인증을 진행합니다.

인증 완료 확인

라이선스 상태가 "활성화됨"으로 표시되는지 확인합니다.

📋 라이선스 유형

  • 단일 라이선스: 한 대의 PC에서만 사용 가능
  • 플로팅 라이선스: 네트워크를 통해 여러 PC에서 공유 사용
  • 교육용 라이선스: 교육 기관 전용 라이선스
  • 평가용 라이선스: 30일 제한 평가판

⚠️ 라이선스 관련 주의사항

  • 라이선스 키는 분실하지 않도록 안전하게 보관
  • 하드웨어 변경 시 라이선스 재등록 필요
  • 불법 복제 사용 시 법적 문제 발생 가능
  • 라이선스 만료 전 갱신 필요

💡 라이선스 관리 팁

  • 라이선스 정보를 별도 파일로 백업
  • 만료일 알림 설정 활용
  • 여러 PC 사용 시 플로팅 라이선스 고려
  • 정기적인 라이선스 상태 확인

🛠️ 초기 설정

기본 환경 설정

언어 설정

Tools → Options → Environment → Language에서 한국어를 선택합니다.

작업 폴더 설정

프로젝트 저장 기본 경로를 설정합니다. (예: C:\GX_Works_Projects\)

자동 저장 설정

자동 저장 간격을 설정하여 작업 내용을 보호합니다. (권장: 5분)

화면 레이아웃 설정

개인 선호에 맞게 툴바, 패널 위치를 조정합니다.

통신 설정

🔌 기본 통신 설정

  • USB 통신: 가장 일반적인 연결 방식
  • Ethernet 통신: 네트워크를 통한 원격 연결
  • Serial 통신: RS-232C를 통한 시리얼 연결

📊 편집기 설정

  • 래더 편집기: 격자 표시, 색상 설정
  • 텍스트 편집기: 폰트, 들여쓰기 설정
  • 함수 블록 편집기: 블록 크기, 연결선 설정
  • SFC 편집기: 스텝 크기, 전환 조건 표시

💡 생산성 향상 설정

  • 자주 사용하는 명령어를 툴바에 추가
  • 단축키 설정으로 작업 속도 향상
  • 코드 템플릿 등록으로 반복 작업 단축
  • 프로젝트 템플릿 생성으로 표준화

🔌 USB 드라이버 설치

드라이버 파일 위치 확인

GX Works2 설치 폴더 내 Driver 폴더에서 USB 드라이버를 찾습니다.

PLC 연결

PLC의 전원을 켜고 USB 케이블로 PC와 연결합니다.

장치 관리자 확인

Windows 장치 관리자에서 "알 수 없는 장치" 또는 "기타 장치"를 확인합니다.

드라이버 수동 설치

해당 장치를 우클릭하여 "드라이버 업데이트"를 선택하고 수동으로 드라이버를 지정합니다.

설치 완료 확인

장치 관리자에서 "Mitsubishi USB Driver"가 정상적으로 표시되는지 확인합니다.

⚠️ 드라이버 설치 주의사항

  • PLC 모델에 맞는 정확한 드라이버 사용
  • Windows 버전 호환성 확인
  • 디지털 서명되지 않은 드라이버 경고 시 계속 진행
  • 설치 후 시스템 재부팅 권장

📋 지원 PLC 모델별 드라이버

  • MELSEC-Q 시리즈: Q 시리즈 전용 USB 드라이버
  • MELSEC-L 시리즈: L 시리즈 전용 USB 드라이버
  • MELSEC-FX 시리즈: FX 시리즈 전용 USB 드라이버
  • MELSEC iQ-R 시리즈: iQ-R 시리즈 전용 USB 드라이버

✅ 설치 확인

프로그램 실행 테스트

GX Works2를 실행하여 정상적으로 시작되는지 확인합니다.

새 프로젝트 생성

File → New Project로 새 프로젝트를 생성해봅니다.

PLC 연결 테스트

Online → Connection Setting에서 PLC 연결을 테스트합니다.

기본 기능 확인

래더 편집, 컴파일, 시뮬레이션 등 기본 기능을 테스트합니다.

✅ 설치 성공 체크리스트

  • GX Works2 프로그램이 정상적으로 실행됨
  • 라이선스가 올바르게 인증됨
  • 새 프로젝트 생성이 가능함
  • PLC와 통신 연결이 정상적으로 작동함
  • 기본 편집 기능들이 모두 작동함
  • 도움말 및 매뉴얼에 접근 가능함

💡 성능 최적화 팁

  • 불필요한 백그라운드 프로그램 종료
  • Windows 업데이트 최신 상태 유지
  • 정기적인 임시 파일 정리
  • 충분한 가상 메모리 설정

🔧 문제 해결

🚫 설치 실패 문제

  • 증상: 설치 중 오류 메시지 발생
  • 원인: 권한 부족, 디스크 공간 부족, 바이러스 백신 간섭
  • 해결책: 관리자 권한으로 실행, 디스크 정리, 바이러스 백신 일시 비활성화

🔑 라이선스 인증 실패

  • 증상: 라이선스 키 입력 후 인증 실패
  • 원인: 잘못된 키 입력, 네트워크 문제, 서버 접속 불가
  • 해결책: 키 재확인, 인터넷 연결 확인, 방화벽 설정 확인

🔌 PLC 연결 실패

  • 증상: PLC와 통신 연결 불가
  • 원인: 드라이버 미설치, 케이블 불량, 포트 설정 오류
  • 해결책: USB 드라이버 재설치, 케이블 교체, 통신 설정 재확인

💻 프로그램 실행 오류

  • 증상: GX Works2 실행 시 오류 발생
  • 원인: .NET Framework 문제, 시스템 파일 손상
  • 해결책: .NET Framework 재설치, 시스템 파일 검사 실행
