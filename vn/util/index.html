유틸리티 목록

  • 📝 Easy Letter Word Counter

    텍스트를 입력하거나 붙여넣으면 글자 수와 단어 수를 즉시 확인합니다!

  • 🪄 Image Compressor

    이미지를 즉시 압축하고 크기를 조절합니다.

  • ✨ Smart Beautifier

    HTML, CSS, JS 코드를 붙여넣으면 자동으로 탭 들여쓰기 형식으로 예쁘게 정리합니다.

  • 🧹 Universal Code Minifier

    모든 주석을 제거하고 JS, CSS, HTML 코드를 압축하여 최소화합니다.

  • 📄 PDF Toolkit

    PDF 병합, 압축, 분할, 암호화, 회전 또는 이미지→PDF 변환을 지원합니다.

  • 🔐 Password Generator

    즉시 강력한 비밀번호를 생성합니다. 원하는 단어를 포함할 수도 있습니다!

  • 🔳 QR Code Generator & Reader

    링크나 텍스트로 QR 코드를 생성하거나, 카메라/이미지 업로드로 QR 코드를 스캔합니다.

  • 🔄 Unit Converter

    길이, 무게, 온도, 속도, 부피, 면적 단위를 즉시 변환합니다.

  • 📸 Social Thumbnail Downloader

    YouTube, X(구 Twitter), TikTok 링크를 붙여넣으면 여러 화질의 썸네일을 미리보기 및 다운로드합니다.

  • 📦 Barcode, ISBN, UPC Generator

    ISBN, UPC, EAN, Code128 등 다양한 바코드를 즉시 생성합니다.

  • 🔎 Text Diff Comparison Tool

    두 텍스트 상자를 붙여넣고 비교 버튼을 누르면, 추가/삭제/변경된 부분을 하이라이트 표시합니다.

  • 🔒 Easy Text Lock & Unlock

    비밀번호 또는 Base64로 텍스트를 간단히 잠그거나 잠금 해제합니다.
    모든 처리는 브라우저에서만 이루어집니다.

  • 🎨 Color Extractor & Converter

    이미지를 업로드하면 주요 색상 팔레트를 즉시 추출하고 HEX, RGB, HSL 값을 변환합니다. 클릭하여 복사하세요!

  • 🔄 Data Format Converter

    JSON, XML, CSV를 붙여넣으면 형식을 변환하고 컬러 뷰로 확인하며 오류를 검사합니다.

  • 🖼️ HTML to Image/PDF

    웹 페이지나 HTML 코드를 PNG/JPG 또는 PDF로 변환합니다. 모든 작업이 브라우저에서 이루어집니다.

  • 🎨 Online Card Maker

    명함, 초대장, 웨딩 카드를 즉시 생성하세요. 템플릿 선택 후 정보를 입력하면 이미지로 저장됩니다!

  • 🔊 Text to Speech (TTS) Converter

    텍스트, 기사, 문서를 자연스러운 음성으로 변환합니다. 브라우저에서 재생하거나 MP3로 다운로드하세요.

  • 🖼️ Image Format Converter

    이미지를 PNG, JPG, JPEG, GIF, BMP, WebP 등 원하는 형식으로 변환합니다.

  • 🖼️ Easy Image Editor

    이미지를 몇 초 만에 크기 조절, 자르기, 워터마크 추가 등 간편히 편집하세요.

  • 📝 Text Summarizer

    긴 텍스트나 기사를 붙여넣고 요약 버튼을 클릭하면 짧은 요약문을 제공합니다!

  • 🔢 CRC Generator

    텍스트나 16진수를 입력하고 CRC 유형을 선택하면 체크섬을 즉시 계산합니다!

  • 🔠 Case Converter

    텍스트를 붙여넣으면 즉시 대소문자를 변환합니다.
    지원: UPPERCASE, lowercase, Capitalize, Title Case, Sentence case.

  • 📝 Quick Memo

    빠르게 메모, 할 일, 아이디어를 적어두세요.
    브라우저 로컬 스토리지에 자동 저장됩니다.

  • 🔀 Text Shuffle & Sort Tool

    여러 줄의 텍스트를 붙여넣으면 즉시 랜덤 섞기, 정렬(A→Z), 역순(Z→A)을 수행합니다!

  • 📈 Compound Interest Calculator

    금액, 이율, 기간을 입력하면 복리로 얼마나 불어나는지 즉시 계산합니다!

  • 🔤 English Pronunciation Converter

    영어 단어 또는 문장을 입력하면 IPA 스타일 발음 기호로 변환합니다.
    공부와 재미를 위한 규칙 기반 변환 (공식 IPA는 아닙니다!)

  • 🏃 BMI Calculator

    키와 몸무게를 입력하면 체질량 지수(BMI)를 즉시 계산합니다!

  • 💸 Simple Finance Calculator

    단리, 저축 예금 성장, 대출 할부 금액을 계산합니다.

  • ⏳ Time Difference & Countdown

    두 날짜 사이의 일, 시간, 분, 초를 계산하거나 실시간 카운트다운을 확인하세요.
    팁: 바로 카운트다운을 원하면 종료 날짜만 설정하세요.

  • 🎂 Age & Birth Year Calculator

    생년월일을 입력하면 정확한 나이, 한국 나이, 다음 생일까지 남은 시간을 보여줍니다.
    또는 나이를 입력하면 출생 연도를 추정할 수 있습니다.

  • ✨ Random Nickname Generator

    한 번에 5개의 랜덤 닉네임을 생성합니다. 스타일을 선택하고 원하는 닉네임을 복사하세요!

  • 🔀 Random Team/Group Generator

    이름을 줄 단위로 입력하고, 팀 수를 선택하면 랜덤으로 팀을 나눕니다!

  • 🔮 Fortune Teller

    오늘의 운세를 확인하려면 버튼을 클릭하세요! (재미용)

  • 🎲 3D Dice Roller

    실제 3D 주사위를 브라우저에서 사실적으로 굴려보세요.

  • 🧹 Text Cleaner

    중복 공백과 줄바꿈을 제거하여 텍스트를 깔끔하게 정리합니다.

  • 🍳 Easy Recipe Finder

    가지고 있는 재료를 입력하거나 선택하면 간편 레시피를 찾아드립니다!
